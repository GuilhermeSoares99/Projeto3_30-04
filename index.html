1. O que é HTML Semântico?

É o uso de tags que descrevem o significado do conteúdo, e não apenas sua aparência. Elas ajudam os leitores de tela a guiar usuários com deficiência visual e facilitam o trabalho dos robôs de busca (SEO).

Vantagens de utilizar a semântica:

  • Acessibilidade: Essencial para leitores de tela.
  • Melhor SEO: Facilita a indexação do Google.
  • Código mais organizado: Manutenção simples e limpa.
